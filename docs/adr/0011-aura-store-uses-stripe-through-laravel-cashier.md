# Aura Store uses Stripe through Laravel Cashier

`aura-cms.com` will use Stripe Checkout and Laravel Cashier for payments, recurring billing, invoices, failed-payment handling, and verified billing webhooks. Aura Store remains the source of truth for products, Project Licenses, entitlements, and Composer access; Stripe is a payment processor rather than the commercial domain or package-distribution platform, and Aura remains responsible for its tax and VAT obligations.
