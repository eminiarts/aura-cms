# aura-cms.com owns Commercial Plugin commerce

The Aura team will build and operate the Commercial Plugin catalog, customer accounts, checkout integration, Project License entitlements, renewals, and authenticated Composer package delivery inside `aura-cms.com` instead of delegating them to a hosted software-sales or private-package platform. This costs more engineering and operational effort, but keeps the customer relationship, commercial rules, and distribution experience under Aura's control.
